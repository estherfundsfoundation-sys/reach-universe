REACH UNIVERSE — WEBSITE FILES

This is a static website, ready for GitHub and Vercel.

Upload every file in this folder to the ROOT of a new GitHub repository:
  - index.html
  - styles.css
  - app.js
  - assets/reach-logo.png
  - assets/reach-branding.png

The assets folder is important. It contains the supplied REACH logo.

After GitHub finishes saving the files, import that repository into Vercel. No API keys are required for this first public website.

Included now:
  - REACH public home page
  - seven live pathway links
  - interactive REACH Mode 72-hour planner
  - Workshop I and Workshop II presentation / kit links
  - Ambassador Headquarters preview and CTA
  - EFF and Instagram links

The later secure features (member sign-in, campus walls, photo uploads, QR check-ins, impact tracking, and private Ambassador Headquarters) need Supabase plus moderation rules before they should be opened to students.
